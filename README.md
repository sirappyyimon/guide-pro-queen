# guide-pro-queen

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不针对任何特定域名或网站，仅作为通用页面资源进行统一管理。

## 项目简介

这是一个静态 HTML 页面的集合仓库。所有页面均为独立文件，不依赖后端服务或外部数据库。你可以直接通过浏览器打开这些 HTML 文件，或将其部署到任何静态托管服务上。

本仓库的主要目的是方便对多个独立页面进行版本控制、归档和分发。

## 目录说明

```
.
├── index.html          # 仓库首页（可选）
├── pages/              # 存放所有 HTML 页面
│   ├── example1.html   # 示例页面 1
│   ├── example2.html   # 示例页面 2
│   └── ...
├── assets/             # 页面所需的静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下存放所有 HTML 页面文件，每个文件对应一个独立页面。
- `assets/` 目录存放页面引用的外部资源，如样式表、脚本和图片。

## 页面归档说明

- 每个 HTML 页面均为自包含的独立文件，可脱离本仓库直接使用。
- 页面之间的引用关系应尽可能避免，如确需引用，请使用相对路径。
- 所有页面均不包含特定网站的标识、链接或品牌信息。
- 归档的页面可能不定期更新或移除，恕不另行通知。
